# 智算令牌集市

这是一个面向中国客户的 AI 大模型 Token 采购咨询页。当前版本是静态演示站，不在线收款、不自动开通模型、不保存 API Key。

## 本地预览

直接打开 `site/index.html` 即可预览。

## 发布到 GitHub Pages

1. 在 GitHub 创建一个新仓库。
2. 把本目录推到仓库的 `main` 分支。
3. 进入仓库 `Settings` -> `Pages`。
4. 在 `Build and deployment` 里选择 `GitHub Actions`。
5. 推送后等待 `Deploy static site to GitHub Pages` 工作流完成。

发布成功后，页面通常会出现在：

`https://<你的用户名>.github.io/<仓库名>/`

## 注意

GitHub Pages 适合展示静态页面和演示项目。正式经营 AI Token 销售、支付、订单、发票、账号交付或 SaaS 服务时，应使用合规的业务服务器或托管平台，并接入实名主体、合同、支付、发票、供应商授权、数据安全和内容安全审核。
